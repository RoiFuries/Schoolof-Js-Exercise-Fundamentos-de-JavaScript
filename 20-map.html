Map

En esta clase veremos cómo transformar un array. El método map() itera sobre los elementos de
un array en el orden de inserción y devuelve array nuevo con los elementos modificados.
Recordemos que MAP siempre nos devuelve un nuevo array! no modifica. pero si se modifica cada uno de los elementos esa modificacion perdura

En el video si se modifico el array personas pero no por .map si no por la función

const pasarAlturasACms = persona => {
persona.altura *= 100
return persona
}

Entonces para que no modifique el primer array en lugar de retornar “persona” retorna un nuevo objeto con la persona desglosada y pisa la altura de persona y ahi es donde pone la multiplicación.

const pasarAlturasACms = persona => {
return {
...persona,
altura: persona.altura *= 100
}
}

Y así es que es un nuevo objeto. o es lo que yo entendí.
